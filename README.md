## 🚀 Git e GitHub para iniciantes

Curso ministrado por **Willian Justen**, onde aprendi os fundamentos do controle de versão com Git e como utilizar o GitHub para gerenciar repositórios, colaborar em projetos e versionar meu código de forma eficiente.

- Conceitos de versionamento
- Comandos essenciais do Git (commit, branch, merge, push, pull)
- Trabalhando com repositórios remotos no GitHub
- Fluxo de trabalho com Pull Requests e Issues
